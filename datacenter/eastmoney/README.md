# eastmoney

东方财富接口封装

## 实现功能

### 选股器

- 按我的默认条件初筛股票
- 自定义我的默认条件参考值筛选股票
- 对筛选结果进行 ROE 排序
- 获取行业列表

### 公司信息

- 所属行业
- 所属概念
- 简介
- 主营业务
- 题材关键字
- 主营构成

### 财务数据

- 获取历史财报信息
- 获取最新财报预约披露时间
- 按类型过滤历史财报：历史一季报，历史中报，历史三季报，历史年报
- 按年份过滤历史财报： xxxx 年-一季报、中报、三季报、年报
- 判断 ROE/EPS/REVENUE/PROFIT 是否逐年递增
- 计算 ROE/EPS 中位数
- 获取今年一季报营收增长比
- 获取历史 PE
- 计算历史 PE 中位数

### 估值

- 获取当前估值状态
- 获取机构评级统计
- 获取盈利预测信息
